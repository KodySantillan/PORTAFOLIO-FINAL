## 🚀 Subir a GitHub Pages
1. Crea repositorio público: `arquitectura-software`
2. Sube TODOS los archivos .html + logo.jpg + foto.jpg
3. Sube la carpeta `archivos/` con los PDFs
4. Settings → Pages → Branch: main → Save
5. Listo en 2-3 minutos

## ✏️ Agregar contenido a una semana (desde GitHub)
1. Abre `semana3.html` en GitHub → editar (lápiz)
2. Busca `Contenido por agregar` y reemplaza con el tema real
3. Busca `archivos/semana3/` y agrega los nombres de tus PDFs
4. Sube los PDFs a la carpeta `archivos/semana3/`
5. En `portafolio.html` quita la clase `coming` de esa semana
